시작 가이드
  1. GitHub 저장소 연결
    사이트 기본 설정에서 GitHub Personal Access Token을 입력하세요
  2. 사이트 디자인 설정
    색상, 폰트, 레이아웃을 원하는 대로 설정하세요
  3. 페이지 만들기
    페이지 관리에서 새 페이지를 추가하고 블록 에디터로 콘텐츠를 작성하세요
  4. 이미지 업로드
    사용할 이미지를 이미지 라이브러리에 업로드하세요
  5. 사이트 배포
    모든 준비가 끝나면 배포 페이지에서 GitHub Pages로 배포하세요
